# Newrare

A modern, responsive landing page presenting Newrare and its innovative mobile applications.

## Overview

Newrare is a mobile app development company showcasing cutting-edge applications available on iOS and Android platforms. This repository contains a static HTML/CSS/JavaScript website that serves as the company's digital presence.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Multi-language Support**: English and French language options with instant switching
- **Modern Animations**: Smooth transitions, parallax scrolling, and gradient animations
- **Accessible Navigation**: Mobile-friendly hamburger menu with smooth scroll anchors
- **Application Showcase**: Featured apps with direct links to App Store and Google Play
- **Contact & Legal**: Contact information, terms and conditions, and company details

## Tech Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with CSS Grid, Flexbox, keyframe animations, and custom properties
- **Vanilla JavaScript**: No dependencies, pure JavaScript for interactivity

## Development

The project is a single-file HTML application with embedded CSS and JavaScript for simplicity and ease of deployment.

### To modify content:
1. Edit the `data-fr` and `data-en` attributes in the HTML for multi-language text
2. Update the corresponding text content in the JavaScript `translations` object

### To modify styling:
1. CSS custom properties are defined in the `:root` selector for easy theming
2. Component-specific styles are organized by section with comments

### To modify functionality:
1. JavaScript is organized into logical functions
2. Main initialization happens in the `DOMContentLoaded` event listener
3. Scroll events handle parallax and animation triggers

## Performance

- Zero external dependencies for optimal loading
- CSS animations use GPU-accelerated transforms
- JavaScript uses passive event listeners for smooth scrolling
- Minimal and optimized code
